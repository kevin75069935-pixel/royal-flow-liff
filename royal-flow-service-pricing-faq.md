# 御澤禮賓商務｜服務、價格規則與 FAQ

本文件可作為 LINE 官方帳號自動回覆、LIFF 預約表單說明、Google Sheet 報價設定與客服話術的基準內容。實際車資仍以車輛檔期、行程細節、司機安排、停車等候、跨區與加值服務確認後的正式報價為準。

## 服務項目

### 1. 機場接送

適用於送機、接機、轉機接待、貴賓抵離台灣等行程。

服務內容：

- 送機服務：接待地址、多點接待、預約機場、航廈、航班資訊、起飛時間。
- 接機服務：預約機場、航廈、航班資訊、抵達時間、接待地址、多點接待。
- 支援機場：桃園國際機場、台北松山機場、台中機場、高雄小港機場。
- 支援航廈：第一航廈、第二航廈、國際線、國內線。
- 可加購：舉牌接機、英文司機、兒童安全座椅、怡雲礦泉水、咖啡 / 茶飲、一次性拖鞋、車內薄毯、企業 Logo 舉牌。

適合客群：

- 企業主管、外賓接待、家庭旅客、藝人名流、VIP 貴賓、長輩與孩童同行旅客。

### 2. 商務接送

適用於企業接待、飯店接送、會議接送、主管用車、貴賓拜訪、展會移動。

服務內容：

- 單趟商務接送。
- 多點接待，最多可增加 4 個接送點。
- 可指定服務等級與加值配備。
- 可協助正式報價單、收據、統編單據或企業月結洽談。

適合客群：

- 企業高層、外商訪客、重要客戶、商務團體、董事會或會議接待。

### 3. 市區商務包時

適用於雙北市區內多點洽公、主管用車、飯店接待、會議往返。

服務內容：

- 3 小時市區商務。
- 8 小時市區商務。
- 客製化商務行程。
- 超過基礎時數依車型每小時計算超時費。
- 跨縣市另依跨區費用設定加價。

### 4. 旅遊包車

適用於半日遊、一日遊、多日旅遊、家庭旅遊、外賓觀光與私人行程。

服務內容：

- 半日包車。
- 一日包車。
- 多日包車。
- 客製化景點安排。
- 可依行程安排車型、司機住宿、司機餐費、停車費與景點特殊費用。

### 6. 結婚禮車

適用於迎娶、宴客、送客、親友接駁、主婚人與貴賓接待。

服務內容：

- 指定高端車型。

## 價格規則

### 1. 機場接送報價

計算基準：

```text
 車型起跳價
+ Google Routes 距離加價
+ Google Routes 車程時間加價
+ 服務項目最低價檢查
+ 加值服務費
+ 增加接送點費
+ 夜間 / 超時 / 等候費
+ 跨區費
= 初步報價
```

備註：

- Google Routes 僅作路線與距離估算。
- 最終車資需由禮賓專員依車輛檔期、司機安排、停車、等候與行程細節確認。
- 接機服務可能因航班延誤、入境時間、停車等候與舉牌需求產生加價。

### 2. 市區商務行程報價

適用於商務接送與商務包車。

計算基準：

```text
 車型 3 小時或 8 小時基礎包時費
+ 超過基礎時數的每小時超時費
+ 跨縣市跨區費
+ 加購服務費
= 市區商務行程報價
```

規則：

- 雙北市區為基準區域。
- 3 小時內使用 3 小時基礎車資。
- 8 小時內使用 8 小時基礎車資。
- 超過 8 小時依車型每小時超時費計算。
- 跨縣市依 `商務跨區費_business_cross_regions` 或 `路線跨區費_pricing_cross_regions` 管理。

### 3. 旅遊包車報價

計算基準：

```text
 每日 8 小時基礎包車費
+ 超時費
+ 司機住宿
+ 司機餐費
+ 跨區費
+ 停車費 / 景點特殊費用
+ 加值服務費
= 旅遊包車報價
```

備註：

- 半日包車、一日包車與多日包車建議分開建立報價規則。
- 景點門票、停車費、船票、司機住宿與餐費可列為另計項目。

### 4. 港口接送報價

計算方式可比照機場接送：

```text
 車型起跳價
+ 距離 / 車程加價
+ 港口接待加值費
+ 增加接送點
+ 跨區費
= 港口接送報價
```

備註：

- 豪華郵輪港口建議保留人工確認機制。
- 離島船班可能受天候影響，建議保留彈性調整時間。

### 5. 結婚禮車 / 展演活動 / 外交禮賓

建議採「專案報價」：

```text
 指定車型費
+ 用車時數
+ 多點接待
+ 司機待命
+ 保密或服儀需求
+ 花束 / 禮品 / 舉牌 / 英文司機
+ 活動窗口協調
= 專案報價
```

備註：

- 建議不完全自動報價，系統提供初步估價後由禮賓專員確認。
- 對高端客群應避免過度碎片化報價，可用「套裝服務」呈現。

## 訂金與付款規則

### 建議規則

- 訂金：建議為總車資 30%，或最低 NT$1,000。
- 車輛保留：收到訂金後保留車輛與司機檔期。
- 尾款：可於出發前匯款或依專員確認方式支付。
- 企業客戶：可依合作狀況提供月結、報價單、收據或統編單據。

### 目前付款方式

- 銀行轉帳
- 銀行轉帳 QR Code
- LINE 回傳匯款資料核對

匯款後請客人回傳：

```text
訂單編號：
匯款金額：
帳號末五碼：
```

- 車型價格調整：改 `車型單價_pricing_car_types`。
- 加值服務調整：改 `加值服務_pricing_addons`。
- 服務最低價調整：改 `服務規則_pricing_services`。
- 跨縣市加價：改 `路線跨區費_pricing_cross_regions` 或 `商務跨區費_business_cross_regions`。
- 商務包時價格：改 `商務包時報價_business_hourly`。

## FAQ

### Q1. 預約後是否立即成立？

送出預約後，系統會先建立訂單並通知禮賓專員。車輛、司機與最終費用需由專員確認後才正式成立。收到訂金後，將為您保留車輛與司機檔期。

### Q2. 報價是否為固定價格？

系統提供的是初步禮賓估價。最終價格會依車輛檔期、路線距離、行程時間、跨區、停車、等候、加值服務與特殊需求確認。

### Q3. 可以增加接送點嗎？

可以。機場接送、港口接送、商務接送、商務包車、結婚禮車、展演活動與外交禮賓皆可增加接送點。共用多點接待最多可增加 4 個點。

### Q4. 多點接送如何收費？

多點接送會依增加點數、繞行距離與等候時間計算。系統可先估算，最終由禮賓專員確認。

### Q5. 接機會追蹤航班嗎？

會。機場接機服務可依航班資訊協助追蹤抵達狀態，但實際接待時間仍需考量入境、行李領取、停車與等候狀況。

### Q6. 如果航班延誤怎麼辦？

航班延誤時，禮賓專員會依航班動態與司機檔期協助調整。若產生長時間等候、停車或行程異動，可能會有額外費用。

### Q7. 可以指定車型嗎？

可以。可選 Lexus LM、Toyota Alphard、Toyota Granvia、Benz V-Class、Benz Vito、Benz Sprinter、VW Crafter 等車型。實際車輛需依檔期確認。

### Q8. 可以指定司機或英文司機嗎？

可以提出需求。英文司機、指定司機、司機服儀、保密行程與企業接待需求，建議提前預約並由專員確認。

### Q9. 是否提供兒童安全座椅？

可以加購兒童安全座椅。請於預約時填寫孩童年齡、身高、體重與座椅數量，方便安排合適座椅。

### Q10. 是否提供礦泉水、拖鞋或點心？

可依服務等級提供。尊榮標準服務含礦泉水、濕紙巾、充電線、薄荷糖與車內清潔。VIP 商務服務可加氣泡水、咖啡、一次性拖鞋、車內薄毯與靜音乘車。

### Q11. 可以開立報價單或收據嗎？

可以。企業客戶可提供正式報價單、收據或統編單據需求。月結需依合作條件另行確認。

### Q12. 如何付款？

目前提供銀行轉帳與銀行轉帳 QR Code。匯款後請於 LINE 回傳訂單編號、匯款金額與帳號末五碼，禮賓專員將協助核對款項。

### Q13. 訂金是多少？

一般建議訂金為總車資 30%，或最低 NT$1,000。實際訂金會依車型、日期、行程與車輛檔期確認。

### Q14. 是否能取消或改期？

可以提出取消或改期需求。若車輛與司機已保留，或特殊車型已安排，可能依取消時間與實際成本酌收費用。

### Q15. 是否支援臨時預約？

可以提出需求，但高端車型、英文司機、舉牌接機、多車調度與特殊禮賓服務建議提前預約，以利安排車輛與司機。

### Q16. 是否提供隱私與保密服務？

提供。企業主管、外賓、藝人名流、私人行程與特殊接待皆可依保密原則處理，並可安排靜音乘車、低調接待與專人窗口。

### Q17. 寰宇商務中心禮賓服務如何計費？

寰宇商務中心禮賓服務可代訂，費用依實際服務內容、機場、貴賓人數與車型套餐確認。商務車與保姆車接送可依不同車型另行報價。

### Q18. 為什麼系統報價後仍需要專員確認？

高端接送服務會受到車輛檔期、司機安排、停車、等候、跨區、特殊禮賓與臨時行程調整影響。系統報價可作快速參考，正式報價仍由禮賓專員確認，確保服務品質與責任清楚。
